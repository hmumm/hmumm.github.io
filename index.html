Projects

Chip8 Interpreter

This program was created to learn about how emulators work. It emulates chip8 instruction set which was used on older computers. It's a fairly simple instructions set so it was a great way to get started on emulation. This uses Mono game to render the screen, and Terminal.GUI for some prompts.

Time Clock Program

This program was used to track how many hours our robotics team members worked on the robot. It used text files to keep track of hours. It allowed these hours to be edited, added, and deleted. It also allowed the administrator to print out the a list of all the students with how many hours they had.
